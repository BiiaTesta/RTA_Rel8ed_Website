Instructions for RTA Tests


Below are the instructions for each test to guide you through the process.


These tests will help determine who will work on special projects, monetary projects, and who will have the opportunity to fill any open positions that become available.

This will also be a good way to show off your skills for the people who are looking for work and looking to get their names in front of important people. We are looking for people to score 90% or higher.

Test 1: Data Labeling



Objective: The goal is to label a given dataset correctly based on the provided guidelines.


Your task is to:

  • Label whether the company header data (name, address, city) matches with the website (Column F “match”).
  • If the company redirects to another URL, specify the redirected URL in (Column G “redirected_url”).

The file contains 600,000 rows. Each of you is expected to go through 500 rows, but there should be no overlap between the labelers. You can assign different ranges using the index column.


Instructions:

  • Open the DataLabeling folder and locate the dataset and guidelines document.
  • Review the guidelines to understand the labeling criteria.
  • Use the data_labeling.py script to label the data. Complete the missing parts of the script if needed.
  • Run the script using: python data_labeling.py --input data/raw_data.csv --output data/labeled_data.csv.
  • Verify your labeled dataset by checking the output CSV to ensure the labels follow the criteria outlined in the guidelines.
  • Submit the labeled labeled_data.csv file along with your completed data_labeling.py script.

Test 2: Database Management



Objective: Create and manage a database to store, query, and retrieve data efficiently.


All resources and instructions for this task are available here: RTA 2024 Test Resources


Instructions:

  • Open the DatabaseManagement folder and locate the db_management.py script.
  • Review the db_schema.sql file to understand the database schema that needs to be created.
  • Implement the functions in db_management.py to create the database, insert data, and run queries.
  • Test the script using: python db_management.py and ensure the database is created, and data is successfully inserted.
  • Run sample queries to retrieve specific records and verify they match the expected results.
  • Submit your completed db_management.py script and any SQL query files you used.

Test 3: Machine Learning



Objective: Build and evaluate a machine learning model using the provided dataset.


All resources and instructions for this task are available here: RTA 2024 Test Resources


Instructions:

  • Open the MachineLearning folder and review the ml_model.py script.
  • Implement a supervised learning model (e.g., classification or regression) based on the type of data provided.
  • Split the dataset into training and testing sets and evaluate the model using metrics like accuracy, precision, recall, or RMSE.
  • Run the model using: python ml_model.py --train data/train.csv --test data/test.csv.
  • Save the trained model using joblib or pickle.
  • Submit your completed ml_model.py script and the saved model file.
